===========================================
自动化公众号写稿系统 - 超级简单启动
===========================================

现在有 3 种方式，选择你喜欢的：


方式 1：双击文件（推荐）⭐⭐⭐⭐⭐
-------------------------------------------
双击：一键启动.bat

会自动：
✓ 启动 Python 服务器
✓ 打开浏览器
✓ 检测服务器状态
✓ 自动跳转到应用


方式 2：打开 HTML
-------------------------------------------
双击：index.html

会自动：
✓ 检测服务器是否启动
✓ 如果已启动 → 自动打开应用
✓ 如果未启动 → 显示启动提示


方式 3：命令行
-------------------------------------------
运行：python app.py

然后打开浏览器访问：
http://localhost:5000
